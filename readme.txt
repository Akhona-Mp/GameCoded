install node,and check the version:
node -v
npm -v

open GameCoded directory on your terminal and run this command:
npm install

If You Need to (Re)Initialize Tailwind (optional)
If you ever need to set up Tailwind from scratch, run:
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

To run the project/run the Development Server:
npm run dev

(Optional) If You Use TypeScript
If you see TypeScript errors, install types:
npm install --save-dev typescript @types/react @types/node

Summary of Required Packages
- next

- react-dom

- autoprefixer
- (optional) typescript, @types/react, @types/node if using TypeScript
